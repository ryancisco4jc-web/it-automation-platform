# IT Automation & AI Operations Platform

## Overview
Enterprise IT automation system covering endpoint management, identity, cloud infrastructure, and AI-driven incident analysis.

---

## 🚀 Capabilities

### MDM Automation (Intune + Jamf)
- Device compliance automation
- Inventory synchronization
- Remediation scripts for failed policies

### AI Log Analysis Engine
- Aggregates logs from Intune, Jamf, Azure, M365
- Detects anomalies (login failures, outages, compliance issues)
- Summarizes incidents using AI
- Suggests root cause + remediation steps

### License Management
- Microsoft 365 license auditing
- Inactive user detection
- Automated license reclamation
- Cost optimization reporting

### Entra ID & Azure Automation
- User lifecycle automation (joiner/mover/leaver)
- Conditional Access audits
- RBAC reviews
- Azure AD cleanup automation

PowerShell | Python | Bash | Microsoft Graph API | Azure | Entra ID | Intune | Jamf | AI APIs
